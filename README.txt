AppleHub final interactive version

Upload all files and folders to the ROOT of the GitHub repository:
- index.html
- style.css
- script.js
- AppleHub.mobileconfig
- assets/
- .nojekyll

Then enable GitHub Pages:
Settings → Pages → Deploy from a branch → main → /(root)
