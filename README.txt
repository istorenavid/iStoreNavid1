iStoreNavid / AppleHub - نسخه تکمیل‌شده با لوگوهای سرویس‌ها

تغییرات این نسخه:
- لوگوی هر وب‌اپ براساس دامنه رسمی همان سرویس نمایش داده می‌شود.
- در صورت خطای سرویس لوگو، favicon مستقیم سایت رسمی استفاده می‌شود.
- صفحات دسته‌بندی، لینک نسخه وب و راهنمای Add to Home Screen حفظ شده‌اند.
- اطلاعات برند و فایل iStoreNavid.mobileconfig حفظ شده‌اند.

انتشار در GitHub Pages:
1) همه فایل‌ها و پوشه‌ها را در ریشه مخزن AppleHub آپلود کنید.
2) GitHub Pages را از Settings > Pages فعال کنید.
3) لینک نصب پروفایل پس از انتشار:
   https://istorenavid.github.io/AppleHub/iStoreNavid.mobileconfig

لوگوها هنگام باز شدن صفحه از منبع favicon دریافت می‌شوند، بنابراین دستگاه کاربر باید به اینترنت متصل باشد.
